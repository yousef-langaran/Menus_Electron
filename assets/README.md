# پوشه Assets

این پوشه برای آیکون‌های برنامه استفاده می‌شود.

## فایل‌های مورد نیاز

برای ساخت فایل اجرایی، به آیکون‌های زیر نیاز دارید:

- `icon.ico` - برای Windows (256x256 یا بزرگتر)
- `icon.icns` - برای macOS (512x512 یا بزرگتر)
- `icon.png` - برای Linux (512x512 یا بزرگتر)

## ایجاد آیکون

می‌توانید از ابزارهای آنلاین برای تبدیل PNG به فرمت‌های مورد نیاز استفاده کنید:

- Windows: https://convertio.co/png-ico/
- macOS: https://cloudconvert.com/png-to-icns
- Linux: PNG با سایز 512x512 کافی است

## جایگزین موقت

اگر آیکون ندارید، می‌توانید خطوط مربوط به icon را از `package.json` حذف کنید یا از یک آیکون پیش‌فرض استفاده کنید.
