# راهنمای شروع سریع

## نصب سریع

```bash
# 1. رفتن به پوشه پروژه
cd Menus_Electron

# 2. نصب وابستگی‌ها
npm install

# 3. اجرای برنامه
npm run dev
```

## استفاده اولیه

### 1. ورود به سیستم
- برنامه را اجرا کنید
- با شماره موبایل و رمز عبور مدیر رستوران وارد شوید
- اطلاعات ورود در حافظه ذخیره می‌شود

### 2. ثبت سفارش
- محصولات را از لیست انتخاب کنید
- اطلاعات مشتری را وارد کنید
- نوع سفارش (داخل سالن/بیرون‌بر) را انتخاب کنید
- در صورت نیاز، پرینترها را برای چاپ انتخاب کنید
- دکمه "ثبت سفارش" را بزنید

### 3. کار آفلاین
- در صورت قطع اینترنت، برنامه به صورت خودکار به حالت آفلاین می‌رود
- سفارشات در حافظه محلی ذخیره می‌شوند
- هنگام اتصال مجدد، سفارشات به صورت خودکار همگام‌سازی می‌شوند

### 4. چاپ رسید
- در صفحه ثبت سفارش، پرینترهای مورد نظر را انتخاب کنید
- پس از ثبت سفارش، رسید به صورت خودکار چاپ می‌شود
- می‌توانید چندین پرینتر را انتخاب کنید

## نکات مهم

⚠️ **قبل از استفاده:**
- مطمئن شوید پرینترها به سیستم متصل هستند
- اتصال اینترنت برای اولین بار (برای دریافت منو) لازم است
- پس از دریافت منو، می‌توانید آفلاین کار کنید

✅ **ویژگی‌های کلیدی:**
- کار آفلاین کامل
- همگام‌سازی خودکار
- چاپ همزمان به چند پرینتر
- رابط کاربری ساده

## ساخت برای انتشار

```bash
# ساخت فایل اجرایی
npm run dist
```

فایل‌های خروجی در پوشه `release` قرار می‌گیرند.
