# Menus Electron App

اپلیکیشن Electron برای مدیریت سفارشات رستوران با قابلیت کار آفلاین

## ویژگی‌ها

- ✅ ثبت سفارش مشتری
- ✅ کار آفلاین - ذخیره سفارشات در حالت آفلاین
- ✅ همگام‌سازی خودکار سفارشات آفلاین با سرور
- ✅ کش منو و اطلاعات ورود از سرور
- ✅ چاپ رسید به دو پرینتر همزمان بدون باز شدن پنجره
- ✅ رابط کاربری ساده و کاربردی

## نصب و راه‌اندازی

### پیش‌نیازها

- Node.js 18 یا بالاتر
- npm یا yarn

### نصب

```bash
cd Menus_Electron
npm install
```

### اجرای برنامه در حالت توسعه

```bash
npm run dev
```

این دستور هم React app و هم Electron را اجرا می‌کند.

### ساخت برنامه

```bash
npm run build
```

برای ساخت فایل اجرایی:

```bash
npm run dist
```

## ساختار پروژه

```
Menus_Electron/
├── electron/          # کدهای Electron (Main Process)
│   ├── main.ts        # فایل اصلی Electron
│   ├── preload.ts     # Preload script
│   ├── database/      # مدیریت دیتابیس آفلاین
│   ├── services/      # سرویس‌های همگام‌سازی و چاپ
│   └── utils/         # توابع کمکی
├── src/               # کدهای React (Renderer Process)
│   ├── pages/         # صفحات برنامه
│   ├── store/         # State management (Zustand)
│   └── services/      # سرویس‌های API
└── dist/              # فایل‌های کامپایل شده
```

## استفاده

### ورود به سیستم

1. برنامه را اجرا کنید
2. با شماره موبایل و رمز عبور وارد شوید
3. اطلاعات ورود در حافظه ذخیره می‌شود

### ثبت سفارش

1. از لیست محصولات، محصولات مورد نظر را به سبد خرید اضافه کنید
2. اطلاعات مشتری را وارد کنید
3. نوع سفارش (داخل سالن/بیرون‌بر) را انتخاب کنید
4. در صورت نیاز، پرینترها را برای چاپ رسید انتخاب کنید
5. سفارش را ثبت کنید

### کار آفلاین

- در حالت آفلاین، سفارشات به صورت محلی ذخیره می‌شوند
- هنگام اتصال به اینترنت، سفارشات به صورت خودکار همگام‌سازی می‌شوند
- می‌توانید به صورت دستی از صفحه تنظیمات همگام‌سازی را انجام دهید

### چاپ رسید

1. در صفحه ثبت سفارش، پرینترهای مورد نظر را انتخاب کنید
2. پس از ثبت سفارش، رسید به صورت خودکار به پرینترهای انتخاب شده چاپ می‌شود
3. چاپ بدون باز شدن پنجره انجام می‌شود

## تنظیمات API

برنامه به صورت پیش‌فرض از آدرس `https://apimenu.promal.ir/api/v1` استفاده می‌کند.

برای تغییر آدرس API، فایل `src/services/api.ts` را ویرایش کنید.
